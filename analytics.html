§ 07 สุขภาพรายวัน & การเติบโต

ข้อมูลสุขภาพและการเติบโตที่ดึงจากใบหน้าเล้า — แยกตายจริง/คัด·รูปแบบเช้า/เย็น + น้ำหนัก/ADG/FCR รายสัปดาห์เทียบ Ross 308 · (น้ำ:อาหาร + อาหารหก/สูญเปล่า ย้ายไปแท็บ "อาหาร vs STD")

7A · แยก "ตายจริง" vs "คัด" + รูปแบบ เช้า/เย็น

คัด = การจัดการเชิงรุก (ดี) · ตายจริง = สูญเสียคุมไม่ได้ (แย่) · ตายเย็นเยอะ = heat stress

7B · น้ำหนัก / ADG / FCR รายสัปดาห์ · เทียบ Ross 308 คละเพศ + กฎ ≥ 4.5× แรกเข้า

ทุกครั้งที่ชั่ง (สัปดาห์) เทียบ ADG จริง กับ Ross 308 ในช่วงเดียวกัน + FCR สะสม กับ Ross 308 · เกณฑ์ฟาร์ม: น้ำหนักต้อง ≥ 4.5× แรกเข้าทุกครั้ง

📊 ข้อมูลดิบ (สำหรับ Excel / BI)

ตัวเลขล้วน เปิดด้วย Excel / Google Sheets / Power BI ได้

คู่มือใช้งาน

  1. เปิดเบราว์เซอร์ Chrome / Edge / Safari / Firefox ใช้ได้หมด
  2. ลากไฟล์ใบหน้าเล้า .xlsx มาวางในกล่อง upload — ทำได้ทีละหลายไฟล์
  3. คลิกแท็บ ตามที่ต้องการดู — คำนวณใน browser ทั้งหมด ไม่ส่งข้อมูลไปไหน
  4. Export PDF/Word — เปิดหน้าต่างใหม่พร้อมรายงานจัดหน้าเรียบร้อย กดพิมพ์/เซฟ

⚠ ถ้า Export ไม่ทำงาน

1. กรณีกดแล้วไม่เห็นไฟล์ — เช็คโฟลเดอร์ Downloads ของเบราว์เซอร์
2. กรณี popup ถูกบล็อก (PDF) — กด 🔒 ที่ address bar → อนุญาต pop-up
3. กรณีเปิดไฟล์โดยตรง (file://) แล้ว Export ไม่ได้ — ใช้ลิงก์ holisticprimary.github.io/Farm-Analytics แทน
4. กรณี Safari Private Mode — เปลี่ยนเป็น Chrome / Edge หรือเปิดในโหมดปกติ
